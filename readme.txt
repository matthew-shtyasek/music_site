Для запуска проекта требуется открыть консоль в корне проекта (в директории, где находится данный файл),
затем ввести туда следующую команду (кавычки обязательны): "venv/Scripts/activate"

Далее требуется создать пользователя и базу данных в PostgreSQL:
'NAME': 'contest'
'USER': 'technology'
'PASSWORD': '123456'
'HOST': 'localhost'
'PORT': '5432'

Импортировать данные теперь можно при помощи команды (виртуальное окружение из первого шага должно быть активировано):
python -Xutf8 loaddata 'music_site/fixtures/final.json'

Для полноценной работы проекта потребуются сервер RabbitMQ (в проекте используется Celery) и сервер Redis.

Для запуска Celery нужно ввести команду:
celery -A music_site worker -l info -P solo

Для запуска проекта нужно ввести команду:
python manage.py runserver

Для запуска кластеризации пользователей по музыкальным предпочтениям следует ввести следующую команду:
python manage.py clustering --weights 0.5,0.25,0.25 --count-clusters 2

Логин и пароль от учётной записи администратора:
admin
123456
